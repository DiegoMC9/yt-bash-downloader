# yt-bash-downloader
--------------------
## Linux Only
--------------------
### Dependencies: youtube-dl ffmpeg curl
#### Get them with:
- **APT** apt-get install youtube-dl ffmpeg curl
- **PACMAN** pacman -S youtube-dl ffmpeg curl
---------------------
### Usage: 
./get-mp3.sh  this is a search
### Options
./get-mp3.sh --set-directory [path to your music directory]
